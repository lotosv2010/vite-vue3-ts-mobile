## 快速启动 | Quick Start

```shell
# 克隆项目
git clone https://git@github.com:lotosv2010/vite-vue3-ts-mobile.git
# 进入项目目录
cd vite+vue3+ts
# 安装依赖 （推荐pnpm）
pnpm install  / yarn
# 启动服务
pnpm dev  / yarn dev
# 打包
pnpm build  / yarn build
# 本地预览打包后的dist文件
pnpm preview / yarn preview
```
